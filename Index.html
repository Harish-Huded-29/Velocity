HOW IT WORKS
Zero Proxy
01
Auth Token
Apps Script hands the browser a short-lived OAuth token — file bytes never touch the server.
02
Direct CDN
Browser fetches straight from Google's CDN via HTTP Range headers for maximum throughput.
03
5× Parallel
Five chunks download simultaneously, saturating your bandwidth instead of single-streaming.
04
Assembly
Chunks are stitched into a Blob in-memory and saved with the original filename.
